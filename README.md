# Online Eğitim Sistemi (C)

Bu proje C dili kullanılarak geliştirilmiş,
menü tabanlı bir online eğitim sistemidir.

## Özellikler
- 1D ve 2D diziler
- Char dizileri
- Matris yapıları
- For ve while döngüleri
- Menü tabanlı kullanıcı etkileşimi

## Derleme ve Çalıştırma
```bash
gcc main.c -o proje
./proje
